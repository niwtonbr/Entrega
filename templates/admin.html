🎨 Identidade Visual (CSS)

Cores Globais Variáveis

Para alterar a cor principal do site, mude as variáveis no topo do CSS do index.html e relatorios.html.

--honda-red: #cc0000; (Vermelho Principal) --honda-blue: #0723f9; (Azul Detalhes) --bg-dark: #0a0a0a; (Fundo Escuro) --neon-green: #39FF14; (Verde Neon)
Animação de Flutuar (Logo e Botões)

Altere a altura do movimento (Y) no @keyframes float.

@keyframes float { 0%, 100% { transform: translateY(0px); } 50% { transform: translateY(-10px); } /* Menor = menos movimento */ }
Efeito de Vidro (Glassmorphism)

Para deixar os cards mais transparentes ou mais own-blur (desfoque):

backdrop-filter: blur(20px); background: rgba(20, 20, 20, 0.7);

📝 Tela de Cadastro (index.html)

Adicionar Novo Modelo de Moto

Procure pela tag <select id="modelo"> e adicione uma nova opção:

<option value="NOVO MODELO">NOME DO MODELO</option>
Adicionar Novo Vendedor

Procure pela tag <select id="vendedor"> e adicione o nome:

<option value="NOME DO VENDEDOR">NOME DO VENDEDOR</option>
Botão de Enviar (Cores e Texto)

Altere a classe .btn-enviar para mudar a cor do gradiente e o texto dentro da tag <button>.

Atualização do Telão (Sincronismo)

No final do script, altere o tempo de atualização (em milissegundos):

setInterval(buscarNoBanco, 30000); // 30000 = 30 segundos

📊 Relatórios (relatorios.html)

Top 3 Destaques (KPIs)

Para mudar de Top 3 para Top 5, procure a função processarDados e altere o slice:

.slice(0, 3); // Mude o 3 para 5
Cores dos Gráficos (Barras e Pizza)

No final do código, procure pela constante colors e troque os códigos HEX:

const colors = ['#FF0055', '#00FFCC', ...];
Altura da Tabela (Scroll)

Para a tabela de ranking ficar maior ou menor, altere o max-height da .scroll-area:

.scroll-area { max-height: 450px; }

⚙️ Servidor (app.py)

Mudar o Nome do Banco de Dados

O banco é gerenciado pelo Render via variável de ambiente. Para mudar a tabela, altere a função init_db().

Adicionar Novo Campo (Ex: Telefone)

Você deve fazer 3 passos:

1. Adicionar coluna no SQL (init_db)
2. Adicionar input no HTML (index.html)
3. Adicionar campo no INSERT (função salvar)
Rotas de Acesso (URLs)

Para mudar o endereço de uma página, altere o @app.route:

@app.route('/admin') -> muda para '/gerencia'

📺 Telão Linux Mint (Python)

Sincronização com Site

Altere a URL no topo do script do telão se mudar o domínio do Render:

API_URL = "https://seusite.onrender.com"
Cores das Linhas (Cores de Motos)

Procure pelo dicionário mapa_cores. Você pode alterar o texto da esquerda e a cor da direita.

"AZUL": ("white", "#0000FF"), # ("Cor da Letra", "Cor do Fundo")
Tempo de Atualização

No final do código, altere o tempo do after:

self.root.after(10000, self.loop) // 10 seg

☁️ Nuvem (Render / GitHub)

Instalar Nova Biblioteca

Sempre adicione o nome da biblioteca no arquivo requirements.txt na raiz do GitHub.

Comando de Inicialização

No painel Render $\rightarrow$ Settings $\rightarrow$ Start Command:

gunicorn app:app
DICA DE OURO: Sempre que alterar o app.py ou requirements.txt, faça o Manual Deploy $\rightarrow$ Clear Cache and Deploy para evitar erros de cache.
