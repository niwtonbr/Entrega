🎨 Cores e Estética (CSS)

Variáveis Globais (:root)

Para mudar as cores de todo o site, altere os valores no topo do CSS:

--honda-red: #cc0000;
--neon-green: #39FF14;
--bg-dark: #0f0f0f;
Animação de Flutuar (Logo/Botões)

Altere a altura do movimento no @keyframes float:

50% { transform: translateY(-10px); }
Efeito Glassmorphism (Vidro)

Altere a transparência e o desfoque nos cards:

backdrop-filter: blur(20px);
background: rgba(20, 20, 20, 0.7);

📝 index.html (Cadastro)

Botão Enviar (Aparência)

Altere a cor do gradiente na classe .btn-enviar.

Campos do Formulário

Para adicionar novas opções de modelos ou cores, altere as tags <option> dentro dos <select>.

Função de Envio (JS)

A função enviarDados() controla o envio para o servidor e a limpeza dos campos.

fetch(`${API_URL}/salvar` ...)
Sincronização do Telão

Altere o tempo de atualização automática no final do script:

setInterval(buscarNoBanco, 30000); // 30 seg

📊 relatorios.html (Analytics)

Cores dos Gráficos

As cores das barras e pizza ficam na função renderCharts, dentro do array colors.

const colors = ['#FF0055', '#00FFCC', ...];
Top 3 Destaques (KPIs)

Para mudar a quantidade de destaques, altere o .slice(0, 3) na função processarDados.

Scroll da Tabela (Altura)

Para a tabela ficar mais comprida ou curta, altere o max-height da .scroll-area.

max-height: 450px;

⚙️ app.py (Backend)

Conexão com Banco (PostgreSQL)

A conexão é feita via variável de ambiente DATABASE_URL do Render.

Rotas da API (End-points)

Cada função @app.route é um comando. Se quiser mudar o nome do link, altere o caminho.

/get_historico -> Alimenta relatórios
/salvar -> Salva moto no telão
Estrutura da Tabela (SQL)

Para adicionar novas colunas (ex: Telefone do cliente), altere a função init_db().

☁️ Render & GitHub

Dependências (requirements.txt)

Sempre que instalar uma biblioteca nova no Python, adicione o nome dela neste arquivo.

Comando de Partida (Start Command)

Configurado no painel do Render para ligar o site:

gunicorn app:app
